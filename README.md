## Как использовать

### Сборка с CMake:
```bash
# Создать build директорию
cmake -B build

# Собрать проект
cmake --build build

# Запустить программу
./build/prog
```

### Компиляция отдельного файла:
```bash
g++ head4/task01_temperature_average.cpp -o task01
./task01
```
